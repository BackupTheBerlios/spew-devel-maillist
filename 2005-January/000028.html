Add and option to increase the buffer size in each
iteration.  One suggestion on how to specify this is:

      $ spew -b 1-100k,1k 100m bigfile

Start at 1k for the first iteration.  Each successive
iteration
uses a buffer size 1k larger than the previous.  Repeat
until
you hit a 100k buffer size.
